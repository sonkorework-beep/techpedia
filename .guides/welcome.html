Добро пожаловать

Techpedia — внутренняя база знаний по оборудованию и специализированному ПО.

Структура

  • Главная — краткая информация и история версий.
  • Гайды — статьи из папки .guides/, список берётся из data/guides.json.
  • Софт — каталог файлов/инструкций из data/software.json.

Как добавить новую статью

  1. Создайте файл статьи в .guides/ (например .guides/new-article.html).
  2. Добавьте запись в data/guides.json: id, title, path, tags, keywords.
  3. Обновите теги/ключевые слова — по ним работает поиск на странице Гайды.
